# 简述

在使用谷歌检索过程中，实际上是存在一些语法命令的，使用好语法命令，可以使得搜索结果更加高效和可用性。根据使用频率，分为以下几个部分

# 操作符

操作符本身，是将一些实际命令集成在搜索词 中，使用好操作符可以高效的检索

+ 强制搜- 逻辑非| 逻辑或* 通配符~ 同义词
避免常用词被过滤掉用于过滤 - 号后面的词作用等同与 OR 多词适配所有的可情况同时检索关键词的同义词

实例

强制搜：+how to build blog

逻辑非：MP3-MP4 (只搜索 MP3 而不要 MP4 的搜索结果)

逻辑或：Google hacking | hacks

通配符：Google was founded in *

同义词：~table

**/^ 指数" " 引号.. 范围
表示指数精确搜索数字范围

实例

指数：2^10 or 2**10.

引号：”Google hacking” = “Google hacking 。

范围：spring festival logo 2002..2009 。（限定范围）

# 操作命令

操作命令可以限定搜索对象的文件类型等

ext: 限定搜索指定文件类型作用同 filetype:

filetype: 限定搜索指定文件类型
作用同 ext: 如： filetype:pdf = ext:pdf 。目前文件类型支持如下格式：
Adobe Acrobat PDF (.pdf) Adobe Postscript (.ps) Autodesk DWF (.dwf) Google地球 KML (.kml) Google地球 KMZ (.kmz) Lotus 1-2-3（wk1、wk2、wk3、wk4、wk5、wki、wks、wku） Lotus WordPro (.lwp) MacWrite (.mw) Microsoft Excel (.xls) Microsoft Powerpoint (.ppt) Microsoft Word (.doc) Microsoft Works（wks、wps、wdb） Microsoft Write (.wri) Rich Text Format富文本格式 (.rtf) Shockwave Flash (.swf) 纯文本（ans、txt）

# 图片搜索与影视搜索

filetype: 限定搜索指定文件类型的图片，支持如下格式：

  • filetype:jpg
  • filetype:gif
  • filetype:png
  • filetype:bmp

imagesize: 限定搜索指定尺寸的图片，如: imagesize:800x600

URL 参数：加在图片搜索网页的 URL 后面，可以实现更精细的搜索

  • &imgtype=news 资讯
  • &imgtype=face 脸部特写
  • &imgtype=clipart 剪贴画
  • &imgtype=lineart 素描
  • &imgtype=photo 照片
  • &imgc=gray 黑白图片
  • &imgc=mono 灰阶图片
  • &imgc=color 全彩图片
